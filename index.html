🚀 Pre-Stream Setup Dock

Step 1: Download your personal dock

This tool runs as a local file to ensure your stream settings are saved privately on your computer.

Step 2: Add to OBS or StreamLabs

For OBS: Go to Docks -> Custom Browser Docks. Name it "Checklist" and paste the local path to the file you just downloaded.

For StreamLabs: Open the Layout Editor, add a Website element, and select the local file.
